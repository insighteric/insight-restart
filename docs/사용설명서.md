# Insight Restart 사용설명서

> **Insight Restart**는 개인회생·파산 업무 전 과정을 AI로 처리하는 실무자용 플랫폼입니다.
> 상담 → 신청 → 보정 → 개시/선고 → 변제/면책까지, 서류·거래내역·일정·수임료·경영까지 한 곳에서 관리합니다.
>
> 웹: **https://insight-restart.vercel.app** · 데스크톱 앱(Windows·Mac) 제공

---

## 목차
1. 시작하기 (접속·설치·회원가입·로그인)
2. 화면 한눈에 보기
3. 메뉴별 사용법
4. 관리자 기능 (경영·분납·멤버 권한)
5. 데이터·보안·자동저장
6. 자주 묻는 질문(FAQ) / 문제 해결

## 1. 시작하기

### 1-1. 접속 방법 (3가지)
| 방법 | 설명 |
|------|------|
| **웹 브라우저** | https://insight-restart.vercel.app 접속 (설치 불필요) |
| **데스크톱 앱** | 메뉴 → **다운로드** → Windows(.exe)·macOS(.dmg) 설치 |
| **앱으로 설치(PWA)** | 다운로드 화면의 "지금 바로 앱으로 설치" → 바탕화면 아이콘 생성 |

> 💡 미서명 앱이라 첫 실행 시 경고가 나올 수 있습니다. Windows: "추가 정보 → 실행", macOS: 앱 우클릭 → 열기.

### 1-2. 회원가입
1. 로그인 화면에서 **회원가입** 탭 클릭
2. **사무소명 · 담당자 이름 · 전화번호 · 이메일 · 비밀번호(6자 이상)** 입력
   - 전화번호는 나중에 **아이디 찾기**에 사용됩니다.
3. **회원가입하고 시작** 클릭 → 자동 로그인되어 대시보드로 이동

### 1-3. 로그인 / 아이디·비밀번호 찾기
- **로그인**: 이메일 + 비밀번호 입력
- **아이디(이메일) 찾기**: 로그인 화면 하단 "아이디 찾기" → 이름 + 전화번호 입력 → 가입 이메일을 마스킹(예: `se***@gmail.com`)으로 확인
- **비밀번호 찾기**: "비밀번호 찾기" → 이메일 입력 → 재설정 링크 메일 수신 → 링크 클릭 → 새 비밀번호 설정

### 1-4. 비밀번호 변경 (로그인 후)
**설정·구독** → **계정 · 비밀번호** 카드 → 새 비밀번호 2번 입력 → **비밀번호 변경**

## 3. 메뉴별 사용법

### 3-1. 대시보드
로그인하면 처음 보이는 화면입니다.
- **진행 중 사건 수**, **임박한 기한(D-day)**, **보정 진행률**을 한눈에 확인
- 임박한 기한·최근 사건을 클릭하면 해당 사건으로 이동

### 3-2. 사건 관리
의뢰인별 개인회생/파산 사건을 관리합니다.
1. **사건 관리** 클릭 → 사건 목록 확인
2. 사건을 클릭하면 상세 화면:
   - **단계 타임라인** (상담→서류준비→접수→보정→개시→변제→면책)
   - **채권자 목록·재산 목록** 입력/수정
   - **변제계획 자동 추정** (소득·가구원수 기준)
   - **AI 절차 진단** — 현재 단계에서 점검할 사항 안내
3. 새 사건은 **사건 추가**(또는 의뢰인 화면에서 사건 생성)로 등록

### 3-3. 사건기록부 ⭐(업무일지)
종이 업무일지를 대신하는 사건별 기록장입니다.
1. **사건기록부** 클릭 → 전체 사건이 최근 기록 순으로 표시
2. 사건을 클릭해 펼치면 기록 목록
3. **기록 추가**: 작성자·내용 입력 → 전송. 중요한 건 **📌 특이사항**으로 고정하면 맨 위로
4. **미납 의뢰인**은 자동으로 빨간 **연체/미납** 배지가 표시됩니다.
- 상단 검색창으로 의뢰인·사건번호 검색 가능

### 3-4. 서류 체크리스트 ⭐
접수·보정 서류 약 50종을 사건별로 관리합니다.
1. **서류 체크리스트** 클릭 → 상단에서 **사건 선택**
2. 분류별(신분·소득·채무·재산·거래내역 등)로 서류가 나열됩니다.
3. 각 서류 오른쪽에서 상태 선택: **미비 / 요청 / 완료 / 해당없음**
   - "완료"로 바꾸면 수령일이 자동 기록됩니다.
4. **발급처 바로가기** 링크로 정부24·홈택스·대법원 등 발급 사이트로 이동
5. 공공·행정 서류는 **⚡자동발급** 버튼으로 발급 가능(아래 4-4 참고)
6. **의뢰인 안내문 복사** — 미비·요청 서류만 모아 카톡/문자용 안내문을 클립보드로 복사 → 의뢰인에게 붙여넣기

### 3-5. 의뢰인
1. **의뢰인** 클릭 → 의뢰인 목록
2. **의뢰인 추가**로 연락처·카톡 ID 등 등록
3. 의뢰인을 클릭하면 진행 사건·연락처 확인

### 3-6. 보정명령 ✨ (AI)
법원 보정명령을 쉽게 풀어 의뢰인에게 전달합니다.
1. **보정명령** 클릭 → 보정명령 **원문을 붙여넣기**
2. **AI 분석** 실행 → 항목별로 자동 분해됩니다:
   - 쉬운 말 요약 · 필요 서류 · 처리 절차 · 의뢰인 당부사항
3. 각 항목을 처리하며 **완료 체크**
4. **의뢰인 안내문**을 **카톡·이메일·문자로 공유**(공유 버튼)
5. 필요 시 **보정서(답변서) 초안** 생성
> AI 키 미설정 시에도 규칙기반으로 동작합니다(품질은 AI 설정 시 더 좋아짐).

### 3-7. AI 서류작성 ✨
1. **AI 서류작성** 클릭 → 사건과 서류 종류 선택
   - 개인회생 신청서 · 채권자목록 · 재산목록 · 수입지출목록 · 변제계획안 · 진술서 · 파산/면책신청서 등
2. **AI 작성** 실행 → 초안 생성 → 내용 검토·수정 → **저장**

### 3-8. 거래내역 분석 ✨ (핵심 기능)
통장·카드 거래내역을 분석해 **은행거래도표·소명자료**를 자동 생성합니다.
1. **거래내역 분석** 클릭
2. **① 의뢰인 이름 입력** (본인계좌 간 이체를 자동으로 걸러냅니다)
3. **은행 / 카드 / 증권·코인** 탭에서 거래내역 입력:
   - 표를 복사해 **붙여넣기**, 또는 **파일 선택**으로 PDF·이미지 업로드
   - 텍스트 PDF는 자동 추출, **스캔본(이미지) PDF는 AI가 읽어** 추출
4. **② 기준금액 필터** 설정(은행·카드 각각 — 그 금액 이상만 추출)
5. **③ 분석 모드(선택)**:
   - 특정인 거래 추출(부분일치) · 대출금 사용처 소명(대출일+금액) · 마이너스 통장 추적(시작일+한도) · 하이라이트 색상
6. **분석 시작** → 결과 탭 확인:
   - 요약 · 입출금 전체 · 입금 · 출금 · 원본 · 특정인 대조표 · 대출금 소명표 · 마이너스 통장 · AI 인사이트(편파변제·낭비 의심 등)
7. 각 시트는 **엑셀(CSV) 다운로드**, 원본 PDF는 **기준금액 하이라이트 PDF**로 받기
> 지원 양식: 카카오뱅크·중소기업은행·삼성카드 등은 전용 인식, 그 외 은행도 일반 인식으로 처리됩니다.

### 3-9. 신복·새출발
회생이 **기각·폐지된 사건**을 채무조정으로 연계 관리합니다.
1. **신복·새출발** 클릭
2. 상단 **"기각·폐지 사건 — 미배정"**에서 대상 사건 **대상 추가**
3. 각 건마다 **연계대상**(신복 개인워크아웃/프리워크아웃·새출발기금·회생 재신청) · **상태**(대상선정→상담→접수→진행중→완료/보류) · **특이사항** 기재

### 3-10. 일정·기한
1. **일정·기한** 클릭 → 보정기한·기일·변제납입 일정 확인(D-day 표시)
2. 일정 추가 시 날짜·유형 입력, **카톡/이메일 알림** 토글
3. 완료된 일정은 체크 처리

### 3-11. 변제 계산기
1. **변제 계산기** 클릭 → 소득·가구원수·재산·채무 입력
2. **청산가치 · 가용소득 · 변제율**이 즉시 계산됩니다(기준 중위소득 설정값 반영)

### 3-12. PDF 도구
브라우저에서 바로 처리되는 PDF 유틸리티(파일이 외부로 전송되지 않음):
- 회전 · 병합 · 분할 · 페이지 삭제 · PDF→이미지 · 이미지→PDF

### 3-13. 설정·구독
- **구독 플랜**: Free / Pro / Team 확인·변경(카드 등록)
- **사무소·기준값**: 사무소명, 생계비 비율, 가구원수별 기준 중위소득
- **연동**: Supabase·토스결제·카카오 알림톡·AI·CODEF 연동 상태
- **계정·비밀번호**: 비밀번호 변경, 로그아웃
- **데이터 초기화**: 데모 데이터 리셋

## 4. 관리자 기능 (권한 필요)

관리자(대표) 또는 해당 **권한을 받은 직원**만 사용할 수 있는 메뉴입니다. 권한이 없으면 메뉴가 보이지 않습니다.

### 4-1. 경영 대시보드 🔒
운영·회계 통계를 기간별로 봅니다.
1. **경영 대시보드** 클릭
2. 상단에서 **기간 선택**: 오늘 / 이번주 / 이번달 / 최근 3개월 / 전체 / **직접 지정**
3. 확인 지표:
   - **기간 매출(수금)** — 분납관리에서 완납 처리된 수임료
   - **신규 계약** 수와 약정액
   - **미수금·연체**
   - **월별 매출·신규계약 추이**, **미수금 상위 의뢰인**, **진행 단계 분포**
> 권한에 따라 매출·미수금·계약 항목이 "권한 없음"으로 가려질 수 있습니다.

### 4-2. 분납관리 🔒
수임료 분할납부와 미납을 관리합니다.
1. **분납관리** 클릭 → 미납 합계·연체 요약 확인
2. **분납계획 만들기**: 사건 선택 → 총 수임료·회차 수·시작일 입력 → 자동으로 회차 생성
3. 각 사건을 펼쳐 회차별 **완납 처리** / 회차 추가
4. **연체/미납**은 자동 계산되고, 사건기록부에도 미납 배지로 표시됩니다.

### 4-3. 멤버·권한 🔒
1. **멤버·권한** 클릭 → 사무소 멤버 목록
2. 각 멤버를 **관리자(대표) / 직원**으로 지정·해제
3. 직원에게는 **항목별 권한**(경영 대시보드·매출·미수금·계약·분납관리·멤버관리)을 골라서 부여
   - 관리자는 항상 모든 권한을 가집니다.
   - 마지막 관리자는 직원으로 변경할 수 없습니다(잠금 방지).
> 같은 사무소에 직원을 합류시키는 **초대 기능**은 추후 추가 예정입니다.

### 4-4. 공공서류 자동발급 (CODEF)
서류 체크리스트에서 정부24·홈택스·대법원·4대보험 서류를 자동 발급합니다.
- 지원 서류 옆 **⚡자동발급** 클릭 → 발급되면 **완료 자동 체크 + 기록 남김**
- 현재는 **목업(시연) 상태**이며, CODEF 키를 설정하면 실제 발급으로 전환됩니다.

## 5. 데이터·보안·자동저장

- **자동 저장**: 입력하는 모든 내용은 자동 저장됩니다(별도 저장 버튼 불필요).
- **로그인 시 클라우드 저장**: 로그인하면 데이터가 사무소 계정(Supabase)에 안전하게 저장되어, 어느 기기에서 접속해도 동일하게 보입니다.
- **권한 분리**: 매출·미수금 등 민감 정보는 권한이 있는 계정만 열람합니다.
- **PDF·거래내역 처리**: 거래내역 분석·PDF 도구는 대부분 브라우저 안에서 처리되어 외부로 전송되지 않습니다(스캔본 AI 추출 시에만 AI 서버로 이미지가 전송됩니다).

## 6. 자주 묻는 질문(FAQ) / 문제 해결

**Q. 접속했더니 "불러오는 중…"에서 멈춰요.**
- 새 버전 배포 직후 캐시 문제일 수 있습니다. **Ctrl + Shift + R**(강력 새로고침) 또는 시크릿 창으로 접속하세요.

**Q. 비밀번호 재설정 메일이 안 와요.**
- 스팸함을 확인하고, 잠시 후 한 번만 다시 시도하세요(메일 발송에 짧은 제한이 있습니다).

**Q. 데스크톱 앱 설치 시 "알 수 없는 개발자" 경고가 떠요.**
- 미서명 앱이라 정상입니다. Windows: "추가 정보 → 실행", macOS: 앱 우클릭 → 열기.

**Q. macOS 앱이 안 열려요.**
- 현재 macOS 앱은 **Apple Silicon(M1 이상)** 전용입니다. Intel Mac은 추후 지원됩니다.

**Q. 메뉴(분납관리·경영 대시보드 등)가 안 보여요.**
- 해당 메뉴는 **권한**이 있어야 보입니다. 대표(관리자)에게 권한을 요청하세요.

**Q. AI 기능 품질을 높이려면?**
- 설정에 AI 키(ANTHROPIC_API_KEY)가 연결되어 있으면 보정·서류 작성 품질이 향상됩니다(미설정 시 규칙기반으로 동작).
