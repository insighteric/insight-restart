# Insight Restart 빠른 시작 가이드 (신입용 1페이지)

> 처음 쓰는 분을 위한 5분 요약입니다. 자세한 내용은 앱의 **도움말** 메뉴 또는 `docs/사용설명서.md`를 보세요.

## 1) 로그인
- 웹: **insight-restart.vercel.app** 접속 → 회원가입(사무소명·이름·전화·이메일·비밀번호) → 로그인
- 데스크톱 앱이 필요하면: 메뉴 **다운로드**에서 Windows·Mac 설치

## 2) 기본 업무 흐름 (사건 1건 처리)
1. **의뢰인** 등록 → **사건 관리**에서 사건 생성
2. **서류 체크리스트**에서 사건 선택 → 필요 서류 체크 → **‘의뢰인 안내문 복사’**로 카톡 안내 발송
3. **보정명령**에 법원 원문 붙여넣기 → **AI 분석** → 의뢰인에게 **카톡 전달**
4. **거래내역 분석**에 통장·카드 내역 입력(또는 PDF 업로드) → **분석 시작** → 소명자료·CSV·하이라이트 PDF
5. **AI 서류작성**으로 신청서·변제계획안 등 초안 생성
6. **일정·기한**에 보정기한·기일 등록(D-day·알림)

## 3) 돈/경영 (관리자)
- **분납관리**: 수임료 분납계획 등록·완납 처리 → 미납/연체 자동 표시
- **경영 대시보드**: 기간별 매출·계약·미수금 통계
- **멤버·권한**: 직원에게 항목별 권한 부여(관리자 지정/해제)

## 4) 꼭 기억할 것
- 입력은 **자동 저장**됩니다(저장 버튼 없음).
- 로그인하면 **어느 기기서나 같은 데이터**가 보입니다.
- 🔒 메뉴(분납·경영·멤버)는 **권한**이 있어야 보입니다.
- 막히면 화면 우측 상단 **? (도움말)** 버튼을 누르세요.

## 5) 문제 해결
- 화면이 멈추면 **Ctrl+Shift+R**(강력 새로고침)
- 비밀번호 재설정 메일이 안 오면 **스팸함** 확인 후 잠시 뒤 1회 재시도
